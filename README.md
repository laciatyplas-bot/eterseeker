## eterseeker

**ETERNIVERSE** — interaktywna mapa uniwersum książek *EterSeeker*.

Krótki opis
- Statyczna strona WWW (HTML/CSS/JS) prezentująca bramy, serie i tytuły: *psyche*, *cień*, *wola*, *obfitość* i *pole*.
- Projekt narracyjno-systemowy przygotowany do publikacji jako GitHub Pages.

Jak uruchomić lokalnie
- Otwórz plik `index.html` w przeglądarce (najprościej: dwuklik lub `File -> Open`).
- Alternatywnie uruchom prosty serwer HTTP (w katalogu repo):

	```bash
	python3 -m http.server 8000
	# lub
	npx http-server -c-1 8000
	```

	Następnie odwiedź: http://localhost:8000

Co dalej
- Dodać favicon, manifest PWA i podstawowy skrypt JS dla interaktywnej mapy.
- Rozwijać zawartość i warstwy mapy.
